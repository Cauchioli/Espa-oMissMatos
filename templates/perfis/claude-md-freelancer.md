## Como opero

Atendo múltiplos clientes simultaneamente, cada um com seu contexto, prazo e entregável. O sistema precisa me ajudar a separar contextos sem confundir informações entre clientes.

## Prioridades de uso do sistema

1. Organização por projeto / cliente (pasta `clientes/[nome]/`)
2. Propostas comerciais rápidas e de alta qualidade
3. Onboarding e entrega padronizados para economizar tempo de briefing

## Regras específicas deste perfil

- Sempre perguntar: "Isso é para qual cliente?" antes de criar qualquer entregável
- Nunca misturar contexto de clientes diferentes na mesma resposta
- Usar `/novo-projeto` para iniciar cada novo cliente
- Usar `/proposta-comercial` + `/client-success` como pipeline padrão

## Stack de trabalho

- **Gestão de projetos:** Pasta `clientes/` com subpastas por cliente
- **Vendas:** Proposta → Call → Onboarding
- **Conteúdo pessoal:** Instagram como vitrine de portfólio e autoridade
